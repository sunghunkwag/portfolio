# Portfolio Landing Page

This repository hosts the single-page portfolio landing experience for **Sung Hun Kwag**. The page highlights a premium 3D title reveal, a refined name introduction, and a single call-to-action that directs visitors to the main GitHub profile.

## Features

- High-fidelity 3D slat title animation with cinematic glow and depth.
- Premium slow-reveal name treatment under the main title.
- Responsive, high-impact GitHub call-to-action button with hover and click effects.
- Gray-toned vignette background for a polished presentation.

## Structure

- `index.html` contains the full layout, styles, and animations.
- `docs/` mirrors the static assets for GitHub Pages deployment.

## Local Preview

Open the page locally with a simple HTTP server:

```bash
python -m http.server 8000
```

Then visit `http://127.0.0.1:8000/` in your browser.

## Deployment

This repository is intended for GitHub Pages. Ensure the `docs/` directory (or root, depending on your Pages settings) contains the latest `index.html` when publishing.
